## 📋 What is nixos-config-tui?

nixos-config-tui is a simple text-based tool that lets you look through and compare NixOS configuration files. It runs in a terminal window and helps you understand the details of your NixOS setup without digging through complicated files by hand.

This tool is designed to help you see what changed between different config versions or quickly spot differences in your config files. It works at the config-level, showing the key parts of your NixOS setup with a clean, easy-to-navigate interface.
